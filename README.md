## ✨ Fitur

- 📝 Tulis dan simpan catatan dengan judul
- 🧮 Auto Math — hitung angka langsung dari teks
- 💾 Data tersimpan otomatis di perangkat
- ✏️ Edit dan hapus catatan kapan saja

---

## 📖 Cara Menggunakan

### Mencatat
1. Klik tombol **+** untuk membuka form catatan
2. Ketik judul (opsional)
3. Tulis isi catatan
4. Klik **Simpan**

### Menghitung (Auto Math)

**Cara 1 — Baris Tunggal**

Tulis angka dan operasi langsung di catatan, contoh:
```
1+1
10×5
100÷4
```
Tooltip hasil akan muncul otomatis. Klik **= Hasil** untuk memasukkan hasil, atau **(Hasil)** untuk format kurung. Tidak mau dimasukkan? Abaikan saja.

**Cara 2 — Hitung Berurutan (Belanjaan)**

Tulis daftar item dengan format `nama : angka`, lalu di baris terakhir tulis tanda operasi:
```
Ayam : 15000
Mie  : 7000
+=
```
Operasi yang tersedia: `+=` `−=` `×=` `÷=`

---

## 🗑️ Mengelola Catatan

- Klik **Pilih** untuk masuk mode kelola
- Klik ✏️ untuk edit catatan
- Klik ❌ untuk hapus satu catatan
- Klik **Hapus Semua** untuk menghapus semua catatan
- Klik **Batal** untuk keluar dari mode kelola

## 📥 Instalasi APK

1. Download file **MathNote.apk** dari tombol di atas
2. Buka file APK di HP
3. Izinkan install dari sumber tidak dikenal jika diminta
4. Install selesai!
